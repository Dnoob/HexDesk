# Changelog

## v0.1.0 (2026-03-05)

首个公开版本。

### 功能

- 自然语言对话（流式输出、Markdown 渲染、代码高亮）
- 文件操作（读 / 写 / 搜索 / 列目录）
- Shell 命令执行
- 操作确认机制（目录白名单 + 危险命令拦截）
- 图片识别（多模态输入）
- 会话管理（创建 / 切换 / 删除，SQLite 持久化）
- 设置页面（API Key、模型选择、多 Provider）
- 文档生成（Word / Excel / PDF）
- 定时任务
- Skills / 插件系统
- MCP 协议集成
- 多 LLM Provider 支持（MiniMax / OpenAI / DeepSeek）
- Agent 多步规划与自主执行

### 平台

- Windows (x64) — MSI / NSIS 安装包
- macOS (Apple Silicon) — DMG
